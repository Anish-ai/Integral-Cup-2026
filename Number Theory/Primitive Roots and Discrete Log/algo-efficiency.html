A. Order of an Integer

Before studying generators of multiplicative groups (primitive roots), we must formally establish the cyclical behavior of integers under modular multiplication. We operate within the multiplicative group of integers modulo $n$, denoted as $(\mathbb{Z}/n\mathbb{Z})^\times$, which consists of all integers coprime to $n$.

Definition 1: Multiplicative Order Let $n > 1$ be an integer and $a$ be an integer such that $\gcd(a, n) = 1$. The multiplicative order of $a$ modulo $n$, denoted as $\text{ord}_n(a)$, is the smallest positive integer $k$ such that: $$a^k \equiv 1 \pmod n$$

By Euler's Totient Theorem, we know that $a^{\phi(n)} \equiv 1 \pmod n$, which guarantees that the set of integers $k$ satisfying $a^k \equiv 1 \pmod n$ is non-empty. Therefore, by the Well-Ordering Principle, the minimum positive integer $\text{ord}_n(a)$ always exists.

Theorem 1.1: Order Divides the Exponent Let $\gcd(a, n) = 1$. If $a^k \equiv 1 \pmod n$ for some positive integer $k$, then $\text{ord}_n(a) \mid k$.
Proof.
Let $h = \text{ord}_n(a)$. By the division algorithm, there exist unique integers $q$ and $r$ such that: $$k = qh + r, \quad 0 \le r < h$$ We can rewrite $a^k$ as follows: $$a^k = a^{qh + r} = (a^h)^q \cdot a^r$$ Taking both sides modulo $n$: $$a^k \equiv (1)^q \cdot a^r \equiv a^r \pmod n$$ We are given that $a^k \equiv 1 \pmod n$, which implies $a^r \equiv 1 \pmod n$.

If $r > 0$, this contradicts the minimality of $h$ as the order of $a$ modulo $n$, since $r < h$. Therefore, we must have $r = 0$.
This implies $k = qh$, meaning $h \mid k$. $\blacksquare$
Corollary 1.2 For any $a$ coprime to $n$, $\text{ord}_n(a) \mid \phi(n)$.
Example: Computing the Order Let $n = 7$. $\phi(7) = 6$. The elements of $(\mathbb{Z}/7\mathbb{Z})^\times$ are $\{1, 2, 3, 4, 5, 6\}$. Let us find $\text{ord}_7(2)$.
The possible orders must divide $6$, so they are $1, 2, 3, \text{or } 6$.
  • $2^1 \equiv 2 \pmod 7$
  • $2^2 \equiv 4 \pmod 7$
  • $2^3 \equiv 8 \equiv 1 \pmod 7$
Thus, $\text{ord}_7(2) = 3$. Notice that $3 \mid 6$.

Advanced Addition: The Carmichael Function $\lambda(n)$

While Euler's theorem states that $a^{\phi(n)} \equiv 1 \pmod n$, $\phi(n)$ is not always the smallest universal exponent for the group $(\mathbb{Z}/n\mathbb{Z})^\times$. The smallest universal exponent is given by the Carmichael function, $\lambda(n)$.

Definition 2: The Carmichael Function For a positive integer $n$, $\lambda(n)$ is defined recursively using its prime factorization.

For prime powers:
  • $\lambda(p^\alpha) = \phi(p^\alpha) = p^{\alpha-1}(p-1)$ for an odd prime $p$, or for $p=2$ with $\alpha \le 2$.
  • $\lambda(2^\alpha) = \frac{1}{2}\phi(2^\alpha) = 2^{\alpha-2}$ for $\alpha \ge 3$.
For a general integer $n = p_1^{\alpha_1} p_2^{\alpha_2} \cdots p_k^{\alpha_k}$: $$\lambda(n) = \text{lcm}(\lambda(p_1^{\alpha_1}), \lambda(p_2^{\alpha_2}), \dots, \lambda(p_k^{\alpha_k}))$$
Theorem 1.3: Carmichael's Theorem For any $a$ coprime to $n$, $a^{\lambda(n)} \equiv 1 \pmod n$, and $\lambda(n)$ is the smallest positive integer with this property for all $a \in (\mathbb{Z}/n\mathbb{Z})^\times$.

Algorithmic Note: In competitive programming, evaluating congruences $a^K \pmod n$ for astronomically large $K$ is typically reduced using $K \bmod \phi(n)$. However, substituting $\phi(n)$ with $\lambda(n)$ provides a tighter bound, which is particularly critical when building cryptographic algorithms or optimizing index-calculus reductions.

The condition under which the group is cyclic (meaning it has a primitive root) is exactly when the maximal order equals the group size: $\lambda(n) = \phi(n)$. We will explore the rigorous classification of such moduli in Part B.
