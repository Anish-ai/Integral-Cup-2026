Module 6: Advanced Practice Questions and Exhaustive Solutions

Practice Questions

These 15 problems synthesize core modules, requiring profound algebraic manipulation and algorithmic mapping.

Problem 1: Idempotents and the CRT Find all integers $n \in [0, 30)$ such that $n^2 \equiv n \pmod{30}$. Generalize algorithmically to find the sum of all $x \in [0, 10^9-1]$ such that $x^2 \equiv x \pmod{10^9}$.
Problem 2: Wilson's Theorem Application Let $p \equiv 1 \pmod 4$ be prime. Prove rigorously that $x \equiv \left(\frac{p-1}{2}\right)! \pmod p$ is a valid solution to $x^2 \equiv -1 \pmod p$.
Problem 3: Sum of $p$-th Powers Let $p$ be an odd prime. Prove that $\sum_{i=1}^{p-1} i^p \equiv 0 \pmod{p^2}$.
Problem 4: Glaisher's Theorem via Lucas Let $p$ be prime. Prove the number of entries in the $n$-th row of Pascal's triangle not divisible by $p$ is $\prod_{i=0}^r (n_i + 1)$, where $n_i$ are the base-$p$ digits of $n$.
Problem 5: Structure of Primitive Roots Let $p > 3$ be prime. Prove there are exactly $\phi(p-1)$ primitive roots modulo $p$, and their product is $1 \pmod p$.
Problem 6: Hensel's Exception Modulo $2^k$ Determine the exact number of solutions to $x^2 \equiv 1 \pmod{2^k}$ for $k \ge 3$, and explicitly find their algebraic forms.
Problem 7: Sum of Squares Uniqueness Assume prime $p \equiv 1 \pmod 4$ satisfies $p = x^2 + y^2$. Using $\mathbb{F}_p$, prove this representation is unique up to absolute values and ordering.
Problem 8: Wolstenholme Extension Using polynomial $P(x) = \prod_{i=1}^{p-1} (x-i) \pmod{p^2}$, prove that for $p \ge 5$, $\sum_{i=1}^{p-1} \frac{1}{i^2} \equiv 0 \pmod p$.
Problem 9: Order and Non-existence Prove that the congruence $n \mid 2^n - 1$ has no integer solutions for $n > 1$.
Problem 10: Carmichael Topology Using Korselt's Criterion, prove that every Carmichael number must have at least three distinct prime factors.
Problem 11: Cyclotomy and Orders Prove that $x^4 \equiv -1 \pmod p$ has a solution if and only if $p \equiv 1 \pmod 8$ (or $p=2$).
Problem 12: Fibonacci Sequence Modulo $p$ Prove that for any odd prime $p \neq 5$, the $p$-th Fibonacci number satisfies $F_p \equiv \left(\frac{p}{5}\right) \pmod p$, where $\left(\frac{p}{5}\right)$ is the Legendre symbol.
Problem 13: Lifting the Exponent (LTE) Calculate the exact highest power of $3$ dividing $2^{3^{100}} + 1$. Implement a compact C++ function to compute $v_p(a^n+b^n)$ for odd primes $p$.
Problem 14: Geometry over $\mathbb{F}_p$ Find the exact number of affine solutions to $x^2 + y^2 \equiv 1 \pmod p$ for an odd prime $p$.
Problem 15: Discrete Logarithm algorithm Formulate the Baby-step Giant-step complexity for solving $a^x \equiv b \pmod m$ when $\gcd(a, m) = 1$. Write a strictly optimized C++ implementation.

Solutions

Solution to Problem 1 $n^2 \equiv n \pmod{30} \implies 30 \mid n(n-1)$. Since $\gcd(n, n-1) = 1$, for each prime $p_i \in \{2, 3, 5\}$, either $p_i \mid n$ or $p_i \mid (n-1)$. This provides $2^3 = 8$ systems of congruences via the CRT. Solutions: $\{0, 1, 6, 10, 15, 16, 21, 25\}$.

Modulo $10^9$: $x \equiv \{0,1\} \pmod{2^9}$ and $x \equiv \{0,1\} \pmod{5^9}$. Let the non-trivial solutions be $x_C$ and $x_D$. Since $x_C \equiv 1 \pmod{2^9}, 0 \pmod{5^9}$ and $x_D \equiv 0 \pmod{2^9}, 1 \pmod{5^9}$, their sum $x_C + x_D \equiv 1 \pmod{10^9}$. Thus, sum is $0 + 1 + 10^9 + 1 = 10^9 + 2$.
Solution to Problem 2 $(p-1)! = \left(\prod_{k=1}^{(p-1)/2} k\right) \left(\prod_{k=1}^{(p-1)/2} (p-k)\right)$. Since $p-k \equiv -k \pmod p$, the second product becomes $(-1)^{(p-1)/2} \prod k$. Thus $(p-1)! \equiv (-1)^{(p-1)/2} \left( \left(\frac{p-1}{2}\right)! \right)^2 \pmod p$. By Wilson's, $(p-1)! \equiv -1 \pmod p$. Since $p \equiv 1 \pmod 4$, $\frac{p-1}{2}$ is even, making $(-1)^{(p-1)/2} = 1$. Substitution yields $-1 \equiv x^2 \pmod p$. $\blacksquare$
Solution to Problem 3 Pair $i$ with $p-i$. Evaluate $(p-i)^p + i^p \pmod{p^2}$. By binomial expansion: $(p-i)^p = p^p - \dots + p^2 i^{p-2} - p \cdot p \cdot i^{p-1} - i^p$. Modulo $p^2$, all terms containing $p^k$ ($k \ge 2$) vanish. Thus $(p-i)^p \equiv -i^p \pmod{p^2}$. The pair sums to $0 \pmod{p^2}$. Since $p$ is odd, all elements pair perfectly, yielding $0 \pmod{p^2}$. $\blacksquare$
Solution to Problem 4 By Lucas's Theorem, $\binom{n}{i} \equiv \prod_{j=0}^k \binom{n_j}{i_j} \pmod p$. For the product to be non-zero in $\mathbb{F}_p$, every term must be non-zero. $\binom{n_j}{i_j} \not\equiv 0 \pmod p$ holds iff $0 \le i_j \le n_j$. For each base-$p$ digit position $j$, there are $n_j + 1$ choices for $i_j$. Independent combinatorics gives $\prod (n_j + 1)$ valid entries. $\blacksquare$
Solution to Problem 5 Group $(\mathbb{Z}/p\mathbb{Z})^\times$ is cyclic of order $p-1$. Generators (primitive roots) are of the form $g^k$ where $\gcd(k, p-1) = 1$. The number of such $k$ is exactly $\phi(p-1)$.
Product $P \equiv g^{\sum k} \pmod p$. Sum of integers coprime to $m$ is $\frac{m}{2} \phi(m)$. Here, exponent is $\frac{p-1}{2} \phi(p-1)$. Thus $P \equiv (g^{(p-1)/2})^{\phi(p-1)} \pmod p$. Since $g$ is primitive, $g^{(p-1)/2} \equiv -1 \pmod p$. For $p > 3$, $\phi(p-1)$ is even. $(-1)^{\text{even}} = 1$. $\blacksquare$
Solution to Problem 6 $2^k \mid (x-1)(x+1)$. If $x$ is odd, $x-1$ and $x+1$ are consecutive evens; one is divisible by exactly 2, the other by $2^{k-1}$.
Case A: $x-1 \equiv 0 \pmod{2^{k-1}} \implies x \in \{1, 2^{k-1}+1\}$.
Case B: $x+1 \equiv 0 \pmod{2^{k-1}} \implies x \in \{2^k-1, 2^{k-1}-1\}$.
These give exactly 4 distinct roots for any $k \ge 3$. $\blacksquare$
Solution to Problem 7 Assume $p = a^2+b^2 = c^2+d^2$. Modulo $p$, $a^2 \equiv -b^2 \implies (ab^{-1})^2 \equiv -1$. Similarly $(cd^{-1})^2 \equiv -1$. In field $\mathbb{F}_p$, roots are limited to $\pm 1$. Thus $ab^{-1} \equiv \pm cd^{-1} \implies ad \equiv \pm bc \pmod p$. Using Brahmagupta-Fibonacci: $p^2 = (ac+bd)^2 + (ad-bc)^2 = (ac-bd)^2 + (ad+bc)^2$. If $ad \equiv bc \pmod p$, since $|ad-bc| < p$, $ad=bc$. This forces $ac+bd = p$, yielding $a=c, b=d$. The alternative sign yields $a=d, b=c$. $\blacksquare$
Solution to Problem 8 Let $P(x) = x^{p-1} - S_1 x^{p-2} + \dots + S_{p-3}x^2 - S_{p-2}x + (p-1)!$. Modulo $p$, roots are $1 \dots p-1$, identical to $x^{p-1}-1$. Thus $S_k \equiv 0 \pmod p$ for $1 \le k \le p-2$.
Evaluating $P(p) \pmod{p^3} \implies S_{p-2} \equiv p S_{p-3} \pmod{p^2}$. Identity $S_{p-2}^2 - 2S_{p-1}S_{p-3} = (p-1)!^2 \sum 1/i^2$. Since $S_{p-2} \equiv 0 \pmod{p^2}$, the $S_{p-2}^2$ term vanishes mod $p^4$. Thus $-2S_{p-3} \equiv (p-1)! \sum 1/i^2 \pmod{p^2}$. Because $S_{p-3} \equiv 0 \pmod p$, RHS must be $0 \pmod p$. $\blacksquare$
Solution to Problem 9 Assume $n > 1$ exists. Let $p$ be the smallest prime factor of $n$. We have $n \mid 2^n - 1 \implies p \mid 2^n - 1$, so $2^n \equiv 1 \pmod p$. By group theory, $\text{ord}_p(2) \mid n$. By FLT, $\text{ord}_p(2) \mid p-1$. Thus $\text{ord}_p(2) \mid \gcd(n, p-1)$. Since $p$ is the smallest prime dividing $n$, no prime divides both $n$ and $p-1$. Hence $\gcd(n, p-1) = 1$. This implies $\text{ord}_p(2) = 1 \implies p \mid (2^1 - 1) = 1$, an impossibility. $\blacksquare$
Solution to Problem 10 Assume Carmichael number $n$ has exactly two prime factors, $n = pq$ with $p < q$. By Korselt's, $n$ is square-free and $q-1 \mid n-1$. $n-1 = pq - 1 = p(q-1) + p - 1$. Since $q-1 \mid n-1$ and $q-1 \mid p(q-1)$, it must follow that $q-1 \mid p-1$. However, we assumed $p < q \implies p-1 < q-1$. A larger integer cannot divide a strictly smaller positive integer. Contradiction. Thus $\ge 3$ factors. $\blacksquare$
Solution to Problem 11 $x^4 \equiv -1 \pmod p \implies x^8 \equiv 1 \pmod p$. The order of $x$ divides 8. If order is 1, 2, or 4, then $x^4 \equiv 1$, contradicting $x^4 \equiv -1$. Thus the order is exactly 8. By Lagrange, the order of an element divides group size $p-1$. Thus $8 \mid p-1 \implies p \equiv 1 \pmod 8$. $\blacksquare$
Solution to Problem 12 Binet's Formula: $F_n = \frac{1}{2^n \sqrt{5}} [ (1+\sqrt{5})^n - (1-\sqrt{5})^n ]$. Modulo $p$, expand via Binomial Theorem: $(1+\sqrt{5})^p - (1-\sqrt{5})^p = 2 \sum_{k=0}^{(p-1)/2} \binom{p}{2k+1} (\sqrt{5})^{2k+1}$. Since $\binom{p}{2k+1} \equiv 0 \pmod p$ for $0 < 2k+1 < p$, only the $k = (p-1)/2$ term survives. Term is $2 (\sqrt{5})^p = 2 \cdot 5^{(p-1)/2} \sqrt{5}$. $F_p = \frac{1}{2^p \sqrt{5}} \cdot 2 \cdot 5^{(p-1)/2} \sqrt{5} = \frac{2}{2^p} 5^{(p-1)/2}$. By FLT, $2^p \equiv 2 \pmod p$. By Euler's Criterion, $5^{(p-1)/2} \equiv \left(\frac{5}{p}\right) \pmod p$. By Quadratic Reciprocity, $\left(\frac{5}{p}\right) = \left(\frac{p}{5}\right)$. $\blacksquare$
Solution to Problem 13 LTE Lemma for odd primes: $v_p(x^n+y^n) = v_p(x+y) + v_p(n)$ if $n$ is odd and $p \mid (x+y), p \nmid x, y$. Here $x=2, y=1, n=3^{100}, p=3$. $p \mid (2+1)$. $v_3(2^{3^{100}} + 1) = v_3(3) + v_3(3^{100}) = 1 + 100 = 101$.

Solution to Problem 14 Total solutions $N = \sum_{x=0}^{p-1} \left(1 + \left(\frac{1-x^2}{p}\right)\right) = p + \sum_{x=0}^{p-1} \left(\frac{1-x^2}{p}\right)$. Using finite field character sums, $\sum_{x=0}^{p-1} \left(\frac{ax^2+b}{p}\right) = - \left(\frac{a}{p}\right)$. Here $a = -1$, so sum evaluates to $-(-1)^{(p-1)/2}$. Thus, the number of affine solutions is exactly $p - (-1)^{(p-1)/2}$. $\blacksquare$
Solution to Problem 15 Rewrite $x = i \cdot n - j$, where $n = \lceil \sqrt{m} \rceil$. Then $a^x \equiv b \implies a^{in} \equiv b a^j \pmod m$. Compute all $b a^j$ for $0 \le j < n$ and hash them. Iterate $i$ from $1$ to $n$ calculating $a^{in}$ to find a match. $O(\sqrt{m})$ time and space. 
using l = long long;
#include <map>
#include <cmath>
l p(l x, l y, l m) {
    l r=1; x%=m;
    while(y) { if(y&1) r=r*x%m; x=x*x%m; y>>=1; }
    return r;
}
l b(l a, l c, l m) {
    l n = std::sqrt(m) + 1, e = 1;
    std::map<l, l> v;
    for (l i = 0; i < n; ++i) {
        v[e] = i; e = e * a % m;
    }
    l f = p(a, m - 2, m), g = p(f, n, m), h = c;
    for (l i = 0; i < n; ++i) {
        if (v.count(h)) return i * n + v[h];
        h = h * g % m;
    }
    return -1;
}

Congruences and Modular Arithmetic

Module 6: Advanced Practice Questions and Exhaustive Solutions

Practice Questions

These 15 problems synthesize core modules, requiring profound algebraic manipulation and algorithmic mapping.

Problem 1: Idempotents and the CRT Find all integers $n \in [0, 30)$ such that $n^2 \equiv n \pmod{30}$. Generalize algorithmically to find the sum of all $x \in [0, 10^9-1]$ such that $x^2 \equiv x \pmod{10^9}$.
Problem 2: Wilson's Theorem Application Let $p \equiv 1 \pmod 4$ be prime. Prove rigorously that $x \equiv \left(\frac{p-1}{2}\right)! \pmod p$ is a valid solution to $x^2 \equiv -1 \pmod p$.
Problem 3: Sum of $p$-th Powers Let $p$ be an odd prime. Prove that $\sum_{i=1}^{p-1} i^p \equiv 0 \pmod{p^2}$.
Problem 4: Glaisher's Theorem via Lucas Let $p$ be prime. Prove the number of entries in the $n$-th row of Pascal's triangle not divisible by $p$ is $\prod_{i=0}^r (n_i + 1)$, where $n_i$ are the base-$p$ digits of $n$.
Problem 5: Structure of Primitive Roots Let $p > 3$ be prime. Prove there are exactly $\phi(p-1)$ primitive roots modulo $p$, and their product is $1 \pmod p$.
Problem 6: Hensel's Exception Modulo $2^k$ Determine the exact number of solutions to $x^2 \equiv 1 \pmod{2^k}$ for $k \ge 3$, and explicitly find their algebraic forms.
Problem 7: Sum of Squares Uniqueness Assume prime $p \equiv 1 \pmod 4$ satisfies $p = x^2 + y^2$. Using $\mathbb{F}_p$, prove this representation is unique up to absolute values and ordering.
Problem 8: Wolstenholme Extension Using polynomial $P(x) = \prod_{i=1}^{p-1} (x-i) \pmod{p^2}$, prove that for $p \ge 5$, $\sum_{i=1}^{p-1} \frac{1}{i^2} \equiv 0 \pmod p$.
Problem 9: Order and Non-existence Prove that the congruence $n \mid 2^n - 1$ has no integer solutions for $n > 1$.
Problem 10: Carmichael Topology Using Korselt's Criterion, prove that every Carmichael number must have at least three distinct prime factors.
Problem 11: Cyclotomy and Orders Prove that $x^4 \equiv -1 \pmod p$ has a solution if and only if $p \equiv 1 \pmod 8$ (or $p=2$).
Problem 12: Fibonacci Sequence Modulo $p$ Prove that for any odd prime $p \neq 5$, the $p$-th Fibonacci number satisfies $F_p \equiv \left(\frac{p}{5}\right) \pmod p$, where $\left(\frac{p}{5}\right)$ is the Legendre symbol.
Problem 13: Lifting the Exponent (LTE) Calculate the exact highest power of $3$ dividing $2^{3^{100}} + 1$. Implement a compact C++ function to compute $v_p(a^n+b^n)$ for odd primes $p$.
Problem 14: Geometry over $\mathbb{F}_p$ Find the exact number of affine solutions to $x^2 + y^2 \equiv 1 \pmod p$ for an odd prime $p$.
Problem 15: Discrete Logarithm algorithm Formulate the Baby-step Giant-step complexity for solving $a^x \equiv b \pmod m$ when $\gcd(a, m) = 1$. Write a strictly optimized C++ implementation.

Solutions

Solution to Problem 1 $n^2 \equiv n \pmod{30} \implies 30 \mid n(n-1)$. Since $\gcd(n, n-1) = 1$, for each prime $p_i \in \{2, 3, 5\}$, either $p_i \mid n$ or $p_i \mid (n-1)$. This provides $2^3 = 8$ systems of congruences via the CRT. Solutions: $\{0, 1, 6, 10, 15, 16, 21, 25\}$.

Modulo $10^9$: $x \equiv \{0,1\} \pmod{2^9}$ and $x \equiv \{0,1\} \pmod{5^9}$. Let the non-trivial solutions be $x_C$ and $x_D$. Since $x_C \equiv 1 \pmod{2^9}, 0 \pmod{5^9}$ and $x_D \equiv 0 \pmod{2^9}, 1 \pmod{5^9}$, their sum $x_C + x_D \equiv 1 \pmod{10^9}$. Thus, sum is $0 + 1 + 10^9 + 1 = 10^9 + 2$.
Solution to Problem 2 $(p-1)! = \left(\prod_{k=1}^{(p-1)/2} k\right) \left(\prod_{k=1}^{(p-1)/2} (p-k)\right)$. Since $p-k \equiv -k \pmod p$, the second product becomes $(-1)^{(p-1)/2} \prod k$. Thus $(p-1)! \equiv (-1)^{(p-1)/2} \left( \left(\frac{p-1}{2}\right)! \right)^2 \pmod p$. By Wilson's, $(p-1)! \equiv -1 \pmod p$. Since $p \equiv 1 \pmod 4$, $\frac{p-1}{2}$ is even, making $(-1)^{(p-1)/2} = 1$. Substitution yields $-1 \equiv x^2 \pmod p$. $\blacksquare$
Solution to Problem 3 Pair $i$ with $p-i$. Evaluate $(p-i)^p + i^p \pmod{p^2}$. By binomial expansion: $(p-i)^p = p^p - \dots + p^2 i^{p-2} - p \cdot p \cdot i^{p-1} - i^p$. Modulo $p^2$, all terms containing $p^k$ ($k \ge 2$) vanish. Thus $(p-i)^p \equiv -i^p \pmod{p^2}$. The pair sums to $0 \pmod{p^2}$. Since $p$ is odd, all elements pair perfectly, yielding $0 \pmod{p^2}$. $\blacksquare$
Solution to Problem 4 By Lucas's Theorem, $\binom{n}{i} \equiv \prod_{j=0}^k \binom{n_j}{i_j} \pmod p$. For the product to be non-zero in $\mathbb{F}_p$, every term must be non-zero. $\binom{n_j}{i_j} \not\equiv 0 \pmod p$ holds iff $0 \le i_j \le n_j$. For each base-$p$ digit position $j$, there are $n_j + 1$ choices for $i_j$. Independent combinatorics gives $\prod (n_j + 1)$ valid entries. $\blacksquare$
Solution to Problem 5 Group $(\mathbb{Z}/p\mathbb{Z})^\times$ is cyclic of order $p-1$. Generators (primitive roots) are of the form $g^k$ where $\gcd(k, p-1) = 1$. The number of such $k$ is exactly $\phi(p-1)$.
Product $P \equiv g^{\sum k} \pmod p$. Sum of integers coprime to $m$ is $\frac{m}{2} \phi(m)$. Here, exponent is $\frac{p-1}{2} \phi(p-1)$. Thus $P \equiv (g^{(p-1)/2})^{\phi(p-1)} \pmod p$. Since $g$ is primitive, $g^{(p-1)/2} \equiv -1 \pmod p$. For $p > 3$, $\phi(p-1)$ is even. $(-1)^{\text{even}} = 1$. $\blacksquare$
Solution to Problem 6 $2^k \mid (x-1)(x+1)$. If $x$ is odd, $x-1$ and $x+1$ are consecutive evens; one is divisible by exactly 2, the other by $2^{k-1}$.
Case A: $x-1 \equiv 0 \pmod{2^{k-1}} \implies x \in \{1, 2^{k-1}+1\}$.
Case B: $x+1 \equiv 0 \pmod{2^{k-1}} \implies x \in \{2^k-1, 2^{k-1}-1\}$.
These give exactly 4 distinct roots for any $k \ge 3$. $\blacksquare$
Solution to Problem 7 Assume $p = a^2+b^2 = c^2+d^2$. Modulo $p$, $a^2 \equiv -b^2 \implies (ab^{-1})^2 \equiv -1$. Similarly $(cd^{-1})^2 \equiv -1$. In field $\mathbb{F}_p$, roots are limited to $\pm 1$. Thus $ab^{-1} \equiv \pm cd^{-1} \implies ad \equiv \pm bc \pmod p$. Using Brahmagupta-Fibonacci: $p^2 = (ac+bd)^2 + (ad-bc)^2 = (ac-bd)^2 + (ad+bc)^2$. If $ad \equiv bc \pmod p$, since $|ad-bc| < p$, $ad=bc$. This forces $ac+bd = p$, yielding $a=c, b=d$. The alternative sign yields $a=d, b=c$. $\blacksquare$
Solution to Problem 8 Let $P(x) = x^{p-1} - S_1 x^{p-2} + \dots + S_{p-3}x^2 - S_{p-2}x + (p-1)!$. Modulo $p$, roots are $1 \dots p-1$, identical to $x^{p-1}-1$. Thus $S_k \equiv 0 \pmod p$ for $1 \le k \le p-2$.
Evaluating $P(p) \pmod{p^3} \implies S_{p-2} \equiv p S_{p-3} \pmod{p^2}$. Identity $S_{p-2}^2 - 2S_{p-1}S_{p-3} = (p-1)!^2 \sum 1/i^2$. Since $S_{p-2} \equiv 0 \pmod{p^2}$, the $S_{p-2}^2$ term vanishes mod $p^4$. Thus $-2S_{p-3} \equiv (p-1)! \sum 1/i^2 \pmod{p^2}$. Because $S_{p-3} \equiv 0 \pmod p$, RHS must be $0 \pmod p$. $\blacksquare$
Solution to Problem 9 Assume $n > 1$ exists. Let $p$ be the smallest prime factor of $n$. We have $n \mid 2^n - 1 \implies p \mid 2^n - 1$, so $2^n \equiv 1 \pmod p$. By group theory, $\text{ord}_p(2) \mid n$. By FLT, $\text{ord}_p(2) \mid p-1$. Thus $\text{ord}_p(2) \mid \gcd(n, p-1)$. Since $p$ is the smallest prime dividing $n$, no prime divides both $n$ and $p-1$. Hence $\gcd(n, p-1) = 1$. This implies $\text{ord}_p(2) = 1 \implies p \mid (2^1 - 1) = 1$, an impossibility. $\blacksquare$
Solution to Problem 10 Assume Carmichael number $n$ has exactly two prime factors, $n = pq$ with $p < q$. By Korselt's, $n$ is square-free and $q-1 \mid n-1$. $n-1 = pq - 1 = p(q-1) + p - 1$. Since $q-1 \mid n-1$ and $q-1 \mid p(q-1)$, it must follow that $q-1 \mid p-1$. However, we assumed $p < q \implies p-1 < q-1$. A larger integer cannot divide a strictly smaller positive integer. Contradiction. Thus $\ge 3$ factors. $\blacksquare$
Solution to Problem 11 $x^4 \equiv -1 \pmod p \implies x^8 \equiv 1 \pmod p$. The order of $x$ divides 8. If order is 1, 2, or 4, then $x^4 \equiv 1$, contradicting $x^4 \equiv -1$. Thus the order is exactly 8. By Lagrange, the order of an element divides group size $p-1$. Thus $8 \mid p-1 \implies p \equiv 1 \pmod 8$. $\blacksquare$
Solution to Problem 12 Binet's Formula: $F_n = \frac{1}{2^n \sqrt{5}} [ (1+\sqrt{5})^n - (1-\sqrt{5})^n ]$. Modulo $p$, expand via Binomial Theorem: $(1+\sqrt{5})^p - (1-\sqrt{5})^p = 2 \sum_{k=0}^{(p-1)/2} \binom{p}{2k+1} (\sqrt{5})^{2k+1}$. Since $\binom{p}{2k+1} \equiv 0 \pmod p$ for $0 < 2k+1 < p$, only the $k = (p-1)/2$ term survives. Term is $2 (\sqrt{5})^p = 2 \cdot 5^{(p-1)/2} \sqrt{5}$. $F_p = \frac{1}{2^p \sqrt{5}} \cdot 2 \cdot 5^{(p-1)/2} \sqrt{5} = \frac{2}{2^p} 5^{(p-1)/2}$. By FLT, $2^p \equiv 2 \pmod p$. By Euler's Criterion, $5^{(p-1)/2} \equiv \left(\frac{5}{p}\right) \pmod p$. By Quadratic Reciprocity, $\left(\frac{5}{p}\right) = \left(\frac{p}{5}\right)$. $\blacksquare$
Solution to Problem 13 LTE Lemma for odd primes: $v_p(x^n+y^n) = v_p(x+y) + v_p(n)$ if $n$ is odd and $p \mid (x+y), p \nmid x, y$. Here $x=2, y=1, n=3^{100}, p=3$. $p \mid (2+1)$. $v_3(2^{3^{100}} + 1) = v_3(3) + v_3(3^{100}) = 1 + 100 = 101$.

Solution to Problem 14 Total solutions $N = \sum_{x=0}^{p-1} \left(1 + \left(\frac{1-x^2}{p}\right)\right) = p + \sum_{x=0}^{p-1} \left(\frac{1-x^2}{p}\right)$.
