Congruences and Modular Arithmetic

Module 4: Polynomial Congruences

1. Roots of Polynomials Modulo $m$

When solving algebraic equations in the field of real or complex numbers, a polynomial of degree $n$ has at most $n$ roots. However, in the ring of integers modulo $m$ (denoted $\mathbb{Z}/m\mathbb{Z}$), this fundamental theorem of algebra breaks down if $m$ is composite. This is because $\mathbb{Z}/m\mathbb{Z}$ contains zero divisors.

Example: Consider the polynomial $f(x) = x^2 - 1$. Over $\mathbb{R}$, its roots are merely $\pm 1$. However, modulo $8$, we evaluate $f(x) \equiv 0 \pmod 8$. Testing the elements of the complete residue system, we see that $1^2 \equiv 1$, $3^2 \equiv 9 \equiv 1$, $5^2 \equiv 25 \equiv 1$, and $7^2 \equiv 49 \equiv 1$. Thus, a degree 2 polynomial has 4 distinct roots: $1, 3, 5, 7 \pmod 8$.

To systematically solve $f(x) \equiv 0 \pmod m$ where $m = p_1^{k_1} p_2^{k_2} \dots p_r^{k_r}$, we first solve $f(x) \equiv 0 \pmod{p_i^{k_i}}$ for each prime power factor, and then combine the partial solutions using the Chinese Remainder Theorem.

2. Lagrange's Theorem for Polynomial Roots

While the number of roots can exceed the degree for composite moduli, the classical bound holds strictly when working modulo a prime $p$, as $\mathbb{Z}/p\mathbb{Z}$ (or $\mathbb{F}_p$) forms a field without zero divisors.

Theorem 4.1 (Lagrange's Theorem) Let $p$ be a prime, and let $f(x) = a_n x^n + a_{n-1} x^{n-1} + \dots + a_0$ be a polynomial with integer coefficients such that $a_n \not\equiv 0 \pmod p$. Then the congruence $f(x) \equiv 0 \pmod p$ has at most $n$ mutually incongruent solutions modulo $p$.
Proof (by induction on degree $n$): Base case: For $n = 1$, $f(x) = a_1 x + a_0 \equiv 0 \pmod p$ where $p \nmid a_1$. Since $p$ is prime, $a_1$ has a unique multiplicative inverse modulo $p$. The unique solution is $x \equiv -a_0 (a_1)^{-1} \pmod p$. The number of solutions is $1 \le 1$.

Inductive step: Assume the theorem holds for all polynomials of degree $k$. Consider $f(x)$ of degree $k+1$. If $f(x) \equiv 0 \pmod p$ has no roots, then $0 \le k+1$, and we are done. Suppose it has at least one root, say $r$.

By polynomial division (valid over fields), $f(x) = (x - r)q(x) + R$, where $R$ is a constant. Since $f(r) \equiv 0 \pmod p$, $R \equiv 0 \pmod p$. Thus, $f(x) \equiv (x - r)q(x) \pmod p$, where $q(x)$ is a polynomial of degree $k$ with the same leading coefficient as $f(x)$.

If $s$ is any root of $f(x)$ distinct from $r$, then $f(s) \equiv (s - r)q(s) \equiv 0 \pmod p$. Since $p$ is prime and $s \not\equiv r \pmod p$, $p \nmid (s - r)$. This forces $p \mid q(s)$, meaning $s$ is a root of $q(x) \equiv 0 \pmod p$.

By the inductive hypothesis, $q(x)$ has at most $k$ roots. Adding the root $r$, $f(x)$ has at most $k + 1$ roots. $\blacksquare$

3. Hensel's Lemma

Hensel's Lemma is the number-theoretic equivalent of Newton's method in calculus. It allows us to "lift" a known solution of a polynomial modulo $p^k$ to a solution modulo $p^{k+1}$.

Theorem 4.2 (Hensel's Lemma) Let $f(x)$ be a polynomial with integer coefficients. Suppose there exists an integer $a$ such that $f(a) \equiv 0 \pmod{p^k}$ (with $k \ge 1$) and $f'(a) \not\equiv 0 \pmod p$, where $f'(x)$ is the formal derivative of $f(x)$.

Then there exists a unique integer $t \pmod p$ such that $f(a + t p^k) \equiv 0 \pmod{p^{k+1}}$.
Proof (via Taylor Expansion): We are searching for a lifted root $r = a + tp^k$ satisfying $f(r) \equiv 0 \pmod{p^{k+1}}$. Expanding $f(a + tp^k)$ using the polynomial Taylor series around $a$: $$f(a + tp^k) = f(a) + f'(a)(tp^k) + \frac{f''(a)}{2!}(tp^k)^2 + \dots$$ Since $f(x)$ has integer coefficients, the terms $\frac{f^{(i)}(a)}{i!}$ are strictly integers. We evaluate this expansion modulo $p^{k+1}$. Because $k \ge 1$, for any $i \ge 2$, the exponent $ik \ge 2k \ge k+1$. Thus, $(tp^k)^i \equiv 0 \pmod{p^{k+1}}$ for all $i \ge 2$. The series radically truncates to: $$f(a + tp^k) \equiv f(a) + f'(a)tp^k \pmod{p^{k+1}}$$ We require this expression to be $0 \pmod{p^{k+1}}$. Because $a$ is a root modulo $p^k$, $f(a) = c p^k$ for some integer $c$. Substituting this gives: $$c p^k + f'(a)tp^k \equiv 0 \pmod{p^{k+1}}$$ Dividing the entire congruence by $p^k$ scales the modulus appropriately: $$c + f'(a)t \equiv 0 \pmod p$$ $$f'(a)t \equiv -c \pmod p$$ Since $f'(a) \not\equiv 0 \pmod p$, $f'(a)$ has a unique inverse modulo $p$. We can explicitly solve for $t$: $$t \equiv -c (f'(a))^{-1} \pmod p$$ This $t$ provides the unique lift $r = a + tp^k \pmod{p^{k+1}}$. $\blacksquare$

The following algorithmic snippet implements Hensel's lift. It accepts a polynomial representation (vector of coefficients starting from degree 0) and lifts a known root $a \pmod{p^{k-1}}$ to $\pmod{p^k}$.

using l = long long;
#include <vector>

l p(l x, l y, l m) {
    l r = 1; x %= m;
    while(y) {
        if(y & 1) r = r * x % m;
        x = x * x % m;
        y >>= 1;
    }
    return r;
}

l i(l a, l m) { return p(a, m - 2, m); }

l v(const std::vector<l>& c, l x, l m) {
    l r = 0, q = 1;
    for (l a : c) {
        r = (r + a * q) % m;
        q = (q * x) % m;
    }
    return r;
}

std::vector<l> d(const std::vector<l>& c) {
    std::vector<l> r;
    for (size_t j = 1; j < c.size(); ++j) r.push_back(c[j] * j);
    return r;
}

l h(std::vector<l> c, l a, l pr, l k) {
    l pk_1 = 1; for(int j=1; j<k; ++j) pk_1 *= pr;
    l pk = pk_1 * pr;
    l f = v(c, a, pk);
    if (f == 0) return a;
    l cv = f / pk_1;
    std::vector<l> dc = d(c);
    l df = v(dc, a, pr);
    if (df % pr == 0) return -1; 
    l t = (-cv % pr + pr) % pr * i(df, pr) % pr;
    return a + t * pk_1;
}

4. Advanced Addition: Thue's Lemma and Sums of Squares

Thue's Lemma is an elegant application of the Pigeonhole Principle to modular arithmetic. It is instrumental in proving Fermat's Theorem on sums of two squares, bridging linear congruences with Diophantine equations.

Theorem 4.3 (Thue's Lemma) For any integer $a$ and any prime $p$, there exist integers $x$ and $y$ satisfying $0 < |x| < \sqrt{p}$ and $0 < |y| < \sqrt{p}$ such that: $$x \equiv ay \pmod p$$
Proof: Let $m = \lfloor \sqrt{p} \rfloor + 1$. Note that $m > \sqrt{p}$. Consider the set of integers generated by evaluating the linear expression: $$S = \{u + av \mid 0 \le u \le m - 1, \; 0 \le v \le m - 1\}$$ The total number of pairs $(u, v)$ in this set is $m \times m = m^2$. Since $m > \sqrt{p}$, we have $m^2 > p$.

Because there are $m^2$ pairs but only $p$ possible remainders modulo $p$, the Pigeonhole Principle dictates that at least two distinct pairs must evaluate to the same remainder. Let these distinct pairs be $(u_1, v_1)$ and $(u_2, v_2)$. We have: $$u_1 + a v_1 \equiv u_2 + a v_2 \pmod p$$ Rearranging yields: $$(u_1 - u_2) \equiv a (v_2 - v_1) \pmod p$$ Define $x = u_1 - u_2$ and $y = v_2 - v_1$. This gives $x \equiv ay \pmod p$.

To verify the bounds, note that since $0 \le u_1, u_2 < m$, the difference $|u_1 - u_2| < m$. Since $m = \lfloor \sqrt{p} \rfloor + 1$, the strict inequality gives $|x| \le \lfloor \sqrt{p} \rfloor < \sqrt{p}$. The same logic restricts $|y| < \sqrt{p}$. Finally, if $y = 0$, then $v_1 = v_2$, forcing $u_1 \equiv u_2 \pmod p$, and since $|u_1 - u_2| < p$, $u_1 = u_2$, contradicting that the pairs are distinct. Thus $y \neq 0$ and $x \neq 0$. $\blacksquare$

Application (Fermat's Theorem on Sums of Two Squares): An odd prime $p$ can be written as $p = x^2 + y^2$ if and only if $p \equiv 1 \pmod 4$. If $p \equiv 1 \pmod 4$, $-1$ is a quadratic residue modulo $p$, meaning there exists an $a$ such that $a^2 \equiv -1 \pmod p$. Invoking Thue's Lemma for this $a$, there exist $x, y < \sqrt{p}$ where $x \equiv ay \pmod p$. Squaring both sides yields $x^2 \equiv a^2 y^2 \equiv -y^2 \pmod p$, or $x^2 + y^2 \equiv 0 \pmod p$. Since $0 < x^2 < p$ and $0 < y^2 < p$, their sum satisfies $0 < x^2 + y^2 < 2p$. The only multiple of $p$ strictly between $0$ and $2p$ is $p$ itself, completely proving $x^2 + y^2 = p$.

    return a + t * pk_1;
}

4. Advanced Addition: Thue's Lemma and Sums of Squares

Thue's Lemma is an elegant application of the Pigeonhole Principle to modular arithmetic. It is instrumental in proving Fermat's Theorem on sums of two squares, bridging linear congruences with Diophantine equations.

Theorem 4.3 (Thue's Lemma) For any integer $a$ and any prime $p$, there exist integers $x$ and $y$ satisfying $0 < |x| < \sqrt{p}$ and $0 < |y| < \sqrt{p}$ such that: $$x \equiv ay \pmod p$$
Proof: Let $m = \lfloor \sqrt{p} \rfloor + 1$. Note that $m > \sqrt{p}$. Consider the set of integers generated by evaluating the linear expression: $$S = \{u + av \mid 0 \le u \le m - 1, \; 0 \le v \le m - 1\}$$ The total number of pairs $(u, v)$ in this set is $m \times m = m^2$. Since $m > \sqrt{p}$, we have $m^2 > p$.

Because there are $m^2$ pairs but only $p$ possible remainders modulo $p$, the Pigeonhole Principle dictates that at least two distinct pairs must evaluate to the same remainder. Let these distinct pairs be $(u_1, v_1)$ and $(u_2, v_2)$. We have: $$u_1 + a v_1 \equiv u_2 + a v_2 \pmod p$$ Rearranging yields: $$(u_1 - u_2) \equiv a (v_2 - v_1) \pmod p$$ Define $x = u_1 - u_2$ and $y = v_2 - v_1$. This gives $x \equiv ay \pmod p$.

To verify the bounds, note that since $0 \le u_1, u_2 < m$, the difference $|u_1 - u_2| < m$. Since $m = \lfloor \sqrt{p} \rfloor + 1$, the strict inequality gives $|x| \le \lfloor \sqrt{p} \rfloor < \sqrt{p}$. The same logic restricts $|y| < \sqrt{p}$. Finally, if $y = 0$, then $v_1 = v_2$, forcing $u_1 \equiv u_2 \pmod p$, and since $|u_1 - u_2| < p$, $u_1 = u_2$, contradicting that the pairs are distinct. Thus $y \neq 0$ and $x \neq 0$. $\blacksquare$

Application (Fermat's Theorem on Sums of Two Squares): An odd prime $p$ can be written as $p = x^2 + y^2$ if and only if $p \equiv 1 \pmod 4$. If $p \equiv 1 \pmod 4$, $-1$ is a quadratic residue modulo $p$, meaning there exists an $a$ such that $a^2 \equiv -1 \pmod p$. Invoking Thue's Lemma for this $a$, there exist $x, y < \sqrt{p}$ where $x \equiv ay \pmod p$. Squaring both sides yields $x^2 \equiv a^2 y^2 \equiv -y^2 \pmod p$, or $x^2 + y^2 \equiv 0 \pmod p$. Since $0 < x^2 < p$ and $0 < y^2 < p$, their sum satisfies $0 < x^2 + y^2 < 2p$. The only multiple of $p$ strictly between $0$ and $2p$ is $p$ itself, completely proving $x^2 + y^2 = p$.
