1. Wolstenholme's Theorem

Wolstenholme's Theorem establishes a profound property of binomial coefficients and harmonic sums modulo $p^3$. It is a critical tool in proving higher-order divisibility in combinatorial number theory.

Theorem 5.1 (Wolstenholme's Theorem) For any prime $p \ge 5$, the following congruences hold:
  1. $\binom{2p-1}{p-1} \equiv 1 \pmod{p^3}$
  2. $H_{p-1} = \sum_{i=1}^{p-1} \frac{1}{i} \equiv 0 \pmod{p^2}$
  3. $\sum_{i=1}^{p-1} \frac{1}{i^2} \equiv 0 \pmod p$
Here, fractional equivalence $\frac{1}{i} \pmod m$ signifies the modular inverse of $i$ modulo $m$.
Algebraic Proof: Consider the polynomial over $\mathbb{Z}$: $$P(x) = (x-1)(x-2)\cdots(x-(p-1)) = x^{p-1} - S_1 x^{p-2} + S_2 x^{p-3} - \dots + S_{p-1}$$ where $S_k$ is the $k$-th elementary symmetric polynomial of $\{1, 2, \dots, p-1\}$. Note that $S_{p-1} = (p-1)!$.

Modulo $p$, the roots of $P(x)$ are $1, 2, \dots, p-1$. By Fermat's Little Theorem, the polynomial $x^{p-1} - 1$ shares these exact same roots modulo $p$. Thus, $P(x) \equiv x^{p-1} - 1 \pmod p$, implying that $S_1, S_2, \dots, S_{p-2}$ are all divisible by $p$.

Evaluating $P(p)$ yields: $$P(p) = (p-1)! = p^{p-1} - S_1 p^{p-2} + \dots + S_{p-3} p^2 - S_{p-2} p + (p-1)!$$ Subtracting $(p-1)!$ from both sides and dividing by $p$ gives: $$S_{p-2} = p^{p-2} - S_1 p^{p-3} + \dots + S_{p-3} p$$ Since $p \ge 5$, all terms on the right side containing $p^k$ for $k \ge 2$ vanish modulo $p^2$. Thus, $S_{p-2} \equiv p S_{p-3} \pmod{p^2}$.

Observe that $S_{p-2} = (p-1)! \sum_{i=1}^{p-1} \frac{1}{i} = (p-1)! H_{p-1}$. Since $S_{p-3} \equiv 0 \pmod p$, we have $p S_{p-3} \equiv 0 \pmod{p^2}$. Therefore, $(p-1)! H_{p-1} \equiv 0 \pmod{p^2}$. Because $\gcd((p-1)!, p^2) = 1$, we conclude $H_{p-1} \equiv 0 \pmod{p^2}$, proving (2).

To prove (1), we expand the binomial coefficient: $$\binom{2p-1}{p-1} = \frac{(2p-1)(2p-2)\dots(p+1)}{(p-1)!} = \prod_{i=1}^{p-1} \frac{p+i}{i} = \prod_{i=1}^{p-1} \left(1 + \frac{p}{i}\right)$$ Expanding this product yields: $$\binom{2p-1}{p-1} = 1 + p \sum_{i=1}^{p-1} \frac{1}{i} + p^2 \sum_{1 \le i < j \le p-1} \frac{1}{ij} + \dots$$ The coefficient of $p$ is $H_{p-1}$, which we proved is $0 \pmod{p^2}$, making the $p H_{p-1}$ term vanish modulo $p^3$. The coefficient of $p^2$ is $\frac{S_{p-3}}{(p-1)!}$. Since $S_{p-3} \equiv 0 \pmod p$, the term $p^2 \frac{S_{p-3}}{(p-1)!} \equiv 0 \pmod{p^3}$. All higher power terms inherently contain $p^3$.

Thus, $\binom{2p-1}{p-1} \equiv 1 + 0 + 0 \equiv 1 \pmod{p^3}$. $\blacksquare$

2. Lucas's Theorem

Lucas's Theorem provides a method to compute binomial coefficients modulo a prime $p$ by treating the integers as base-$p$ strings. It acts as an algorithm to drastically reduce the complexity of calculating $\binom{n}{m} \pmod p$ for massive $n, m$.

Theorem 5.2 (Lucas's Theorem) Let $p$ be a prime, and let $n$ and $m$ be non-negative integers. Let their base-$p$ expansions be $n = n_k p^k + \dots + n_1 p + n_0$ and $m = m_k p^k + \dots + m_1 p + m_0$. Then: $$\binom{n}{m} \equiv \prod_{i=0}^k \binom{n_i}{m_i} \pmod p$$ with the convention that $\binom{a}{b} = 0$ if $a < b$.
Proof (Polynomial Expansion): We analyze the polynomial $(1+x)^n$ in the field $\mathbb{Z}/p\mathbb{Z}$. By the Freshman's Dream identity, $(1+x)^p \equiv 1+x^p \pmod p$. By induction, this extends to $(1+x)^{p^i} \equiv 1+x^{p^i} \pmod p$ for all integers $i \ge 0$.

Using the base-$p$ expansion of $n$, we can write: $$(1+x)^n = (1+x)^{n_0 + n_1 p + \dots + n_k p^k} = \prod_{i=0}^k (1+x)^{n_i p^i}$$ Applying the identity, we get: $$(1+x)^n \equiv \prod_{i=0}^k \left(1+x^{p^i}\right)^{n_i} \pmod p$$ Expanding each factor using the standard binomial theorem: $$(1+x)^n \equiv \prod_{i=0}^k \left( \sum_{j_i=0}^{n_i} \binom{n_i}{j_i} x^{j_i p^i} \right) \pmod p$$ To find the coefficient of $x^m$ on the left side (which is $\binom{n}{m}$), we must identify how to form $x^m$ on the right side. Since $m$ has a unique base-$p$ representation $m = m_0 + m_1 p + \dots + m_k p^k$, the only combination of terms from the product that yields $x^m$ is choosing exactly $j_i = m_i$ for every $i$.

Therefore, the coefficient of $x^m$ modulo $p$ is strictly the product of the corresponding coefficients: $$\binom{n}{m} \equiv \prod_{i=0}^k \binom{n_i}{m_i} \pmod p$$ $\blacksquare$

The algorithmic implementation precomputes factorials and their inverses up to $p-1$ in $O(p)$, after which any $\binom{n}{m} \pmod p$ inquiry is resolved in $O(\log_p n)$ recursively.

using t = long long;
#include <vector>

t e(t x, t y, t m) {
    t r = 1; x %= m;
    while(y) {
        if(y & 1) r = r * x % m;
        x = x * x % m;
        y >>= 1;
    }
    return r;
}

t b(t n, t k, t m, const std::vector<t>& f, const std::vector<t>& v) {
    if (k < 0 || k > n) return 0;
    return f[n] * v[k] % m * v[n - k] % m;
}

t u(t n, t k, t m, const std::vector<t>& f, const std::vector<t>& v) {
    if (k == 0) return 1;
    t n0 = n % m, k0 = k % m;
    if (n0 < k0) return 0;
    return b(n0, k0, m, f, v) * u(n / m, k / m, m, f, v) % m;
}

3. Advanced Addition: Babbage's Theorem and Morley's Congruence

While Wolstenholme and Lucas theorems govern base prime moduli behavior, Babbage and Morley derived specialized equivalences that extend combinatorial identities to prime power moduli.

Theorem 5.3 (Babbage's Theorem) For any prime $p$ and non-negative integers $n, m$: $$\binom{np}{mp} \equiv \binom{n}{m} \pmod{p^2}$$ This is an immediate strengthening of Lucas's Theorem specifically for multiples of $p$, elevating the modulus precision from $p$ to $p^2$.
Theorem 5.4 (Morley's Congruence) For any prime $p \ge 5$, the central binomial coefficient evaluates modulo $p^3$ as: $$\binom{p-1}{\frac{p-1}{2}} \equiv (-1)^{\frac{p-1}{2}} 4^{p-1} \pmod{p^3}$$

Morley's Congruence is particularly remarkable as it ties a combinatorial quantity on the left entirely to an exponential expression on the right modulo $p^3$, devoid of generating functions. It sees heavy use in simplifying generating series calculations involving central binomial bounds under field $\mathbb{F}_p$.
