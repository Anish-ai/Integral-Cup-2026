Section C: Dirichlet Convolution and Möbius Inversion

1. The Algebraic Structure of Dirichlet Convolution

Dirichlet convolution is the fundamental binary operation that weaves arithmetic functions together. It bridges the gap between discrete divisors and continuous analytic functions (like the Riemann Zeta function).

Definition 1.1: Dirichlet Convolution Let $f$ and $g$ be two arithmetic functions. Their Dirichlet convolution, denoted $f * g$, is defined as: $$(f * g)(n) = \sum_{d|n} f(d)g\left(\frac{n}{d}\right) = \sum_{ab=n} f(a)g(b)$$
Theorem 1.1: The Commutative Ring of Arithmetic Functions The set of all arithmetic functions $\mathcal{A}$, equipped with point-wise addition $(+)$ and Dirichlet convolution $(*)$, forms a commutative ring.
Proof Sketch To establish $(\mathcal{A}, +, *)$ as a commutative ring, we verify the properties of the convolution operation:

Commutativity: $$(f * g)(n) = \sum_{ab=n} f(a)g(b) = \sum_{ba=n} g(b)f(a) = (g * f)(n)$$ Associativity: $$((f * g) * h)(n) = \sum_{ab=n} (f * g)(a) h(b) = \sum_{ab=n} \left(\sum_{cd=a} f(c)g(d)\right) h(b) = \sum_{cdb=n} f(c)g(d)h(b)$$ $$= \sum_{c(db)=n} f(c) \left(\sum_{xy=db} g(x)h(y)\right) = (f * (g * h))(n)$$ Distributivity: Follows directly from the linearity of finite sums. $\blacksquare$

2. Identity, Constant, and the Möbius Function

Definition 2.1: Identity and Constant Functions The identity function $\epsilon(n)$ under Dirichlet convolution is: $$\epsilon(n) = \begin{cases} 1 & \text{if } n = 1 \\ 0 & \text{if } n > 1 \end{cases}$$ It satisfies $f * \epsilon = f$ for all $f \in \mathcal{A}$.

The constant function $1(n)$ is defined simply as $1(n) = 1$ for all $n \ge 1$.
Definition 2.2: The Möbius Function $\mu(n)$ The Möbius function is the inverse of the constant function $1(n)$ under Dirichlet convolution. Analytically, it is defined as: $$\mu(n) = \begin{cases} 1 & \text{if } n = 1 \\ (-1)^k & \text{if } n = p_1 p_2 \dots p_k \text{ (distinct primes)} \\ 0 & \text{if } n \text{ is not square-free (i.e., } p^2 | n \text{)} \end{cases}$$
Theorem 2.1: Inverse Relationship $$\mu * 1 = \epsilon \iff \sum_{d|n} \mu(d) = \epsilon(n)$$
Proof For $n = 1$, $\mu(1) = 1 = \epsilon(1)$.
For $n > 1$, let $n = p_1^{a_1} p_2^{a_2} \dots p_k^{a_k}$. Since $\mu(d) = 0$ if $d$ is not square-free, the non-zero terms in the sum $\sum_{d|n} \mu(d)$ correspond to the divisors $d$ that are products of distinct primes chosen from $\{p_1, p_2, \dots, p_k\}$.

There are $\binom{k}{r}$ ways to choose a divisor with exactly $r$ distinct prime factors, and for each such divisor, $\mu(d) = (-1)^r$. Thus: $$\sum_{d|n} \mu(d) = \sum_{r=0}^k \binom{k}{r} (-1)^r 1^{k-r}$$ By the Binomial Theorem, this sum evaluates to $(1 - 1)^k = 0 = \epsilon(n)$. $\blacksquare$
Algorithmic Implementation: $O(\sqrt{n})$ $\mu(n)$ Evaluation 
int u(long long n){
if(n==1)return 1;
int c=0;
for(long long i=2;i*i<=n;i++){
if(n%i==0){
n/=i;
if(n%i==0)return 0;
c++;
}}
if(n>1)c++;
return c%2?-1:1;
}

3. The Möbius Inversion Formula

Theorem 3.1: Möbius Inversion Let $f$ and $g$ be arithmetic functions. The classical form states: $$g(n) = \sum_{d|n} f(d) \iff f(n) = \sum_{d|n} g(d)\mu\left(\frac{n}{d}\right)$$
Proof (Via Convolution Ring) The algebraic formulation of the premise is $g = f * 1$. We wish to isolate $f$. By taking the Dirichlet convolution of both sides with $\mu$: $$g * \mu = (f * 1) * \mu$$ Because the convolution operation is associative, we can regroup the right side: $$g * \mu = f * (1 * \mu)$$ By Theorem 2.1, $1 * \mu = \epsilon$. Therefore: $$g * \mu = f * \epsilon$$ $$g * \mu = f$$ Expanding $f = g * \mu$ yields the classical sum $f(n) = \sum_{d|n} g(d)\mu\left(\frac{n}{d}\right)$. $\blacksquare$

4. Advanced Addition: The Mertens Function & The Riemann Hypothesis

The global distribution of the Möbius function encapsulates the randomness of prime factors, tying it irrevocably to the deepest unproven hypothesis in complex analysis.

Definition 4.1: The Mertens Function The Mertens function $M(x)$ is the summatory function of the Möbius function: $$M(x) = \sum_{1 \le n \le x} \mu(n)$$
Theorem 4.1: Analytical Bounds and the Riemann Hypothesis The Mertens conjecture posited that $|M(x)| < \sqrt{x}$ for all $x> 1$. If true, this would have implied the Riemann Hypothesis (RH) because it forces the analytic continuation of $\frac{1}{\zeta(s)}$ to be holomorphic for $\text{Re}(s) > \frac{1}{2}$.

Although Odlyzko and te Riele proved the strict Mertens conjecture false in 1985, the broader asymptotic condition: $$M(x) = O(x^{\frac{1}{2} + \epsilon}) \quad \text{for all } \epsilon > 0$$ remains rigorously equivalent to the Riemann Hypothesis. From an algorithmic standpoint, analyzing $M(x)$ over limits approaching $10^{11}$ forms the core of modern sublinear prefix sum evaluations (which we will explore via the Dirichlet Prefix Sum trick in Section D).
