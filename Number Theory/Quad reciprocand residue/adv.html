Evaluating residuosity is only half the battle. Once we confirm a quadratic residue exists modulo $p$, we must algorithmically extract the root. This falls into the domain of randomized polynomial-time algorithms, forming the backbone of elliptic curve cryptography and advanced competitive programming.

1. The Tonelli-Shanks Algorithm

The Tonelli-Shanks algorithm solves the congruence $x^2 \equiv n \pmod{p}$ for an odd prime $p$. It is fundamentally an extension of the Sylow $2$-subgroup structure within $\mathbb{Z}_p^*$.

Mathematical Foundation First, we factor out powers of $2$ from $p-1$ such that $p-1 = Q \cdot 2^S$, where $Q$ is odd. If $S = 1$ (meaning $p \equiv 3 \pmod{4}$), a direct analytical solution exists: $x \equiv n^{\frac{p+1}{4}} \pmod{p}$.

For $S > 1$, we must search within the cyclic subgroup of order $2^S$. We find an arbitrary quadratic non-residue $z$ (evaluated using Euler's criterion) which acts as a generator for the $2$-Sylow subgroup when raised to the power of $Q$.

We initialize the state variables:
  • $m = S$ (the order of the current subgroup boundary)
  • $c = z^Q \pmod{p}$ (a root of unity of order $2^S$)
  • $t = n^Q \pmod{p}$ (our current equivalence factor)
  • $r = n^{\frac{Q+1}{2}} \pmod{p}$ (our current working root)

Loop invariant: $r^2 \equiv n \cdot t \pmod{p}$. The goal is to drive $t \equiv 1 \pmod{p}$ by iteratively multiplying $t$ by carefully chosen powers of $c$, updating $r$ accordingly. We find the smallest $0 < i < m$ such that $t^{2^i} \equiv 1 \pmod{p}$, update our step size $b = c^{2^{m-i-1}} \pmod{p}$, and adjust the state variables.

Here is the heavily optimized algorithmic execution. Notice that finding a non-residue is done randomly/linearly, which takes expected $O(1)$ time since half of $\mathbb{Z}_p^*$ are non-residues. The loop convergence guarantees worst-case $O(\log^2 p)$ time complexity.

2. Advanced Addition: Cipolla's Algorithm

An alternative to Tonelli-Shanks is Cipolla's Algorithm, which uses arithmetic in quadratic field extensions instead of subgroup reductions. While Tonelli-Shanks relies heavily on modular multiplications bounded by $S$, Cipolla's maintains a rigid $O(\log p)$ bound regardless of $S$.

Theorem 5.1: Cipolla's Root Extraction To solve $x^2 \equiv n \pmod{p}$, find an integer $a$ such that $a^2 - n$ is a quadratic non-residue modulo $p$. Let $\omega = \sqrt{a^2 - n}$. We define the field extension $\mathbb{F}_{p^2} = \mathbb{F}_p[\omega]$.

The square root of $n$ is given directly by computing: $$ x \equiv (a + \omega)^{\frac{p+1}{2}} \pmod{p} $$

The proof relies on Frobenius automorphisms. In $\mathbb{F}_{p^2}$, $(a + \omega)^p = a^p + \omega^p$. By Fermat's Little Theorem, $a^p \equiv a \pmod{p}$. Furthermore, $\omega^p = \omega(\omega^2)^{\frac{p-1}{2}} = \omega(a^2 - n)^{\frac{p-1}{2}} = -\omega$ (since $a^2-n$ is a non-residue). Thus, $(a+\omega)^{p+1} = (a+\omega)(a-\omega) = a^2 - \omega^2 = n$. Taking the square root of both sides gives $(a+\omega)^{\frac{p+1}{2}}$, which must lie in $\mathbb{F}_p$.

3. Advanced Addition: Solovay-Strassen Primality Testing

The structural disparity between the Legendre symbol and the Jacobi symbol over composite numbers gives rise to probabilistic primality testing. Before the widespread adoption of Miller-Rabin, Solovay-Strassen utilized Euler's Criterion to establish a deterministic lower bound on compositeness witnesses.

Definition 5.2: Solovay-Strassen Test Let $n$ be an odd integer to test for primality. Choose a random integer $a$ such that $1 < a < n-1$.

If $\gcd(a, n) > 1$, $n$ is composite. Otherwise, compute the Jacobi symbol $\left(\frac{a}{n}\right)$ using the $O(\log a \log n)$ reciprocity algorithm, and independently compute $a^{\frac{n-1}{2}} \pmod{n}$.

If Euler's criterion fails to hold, that is: $$ a^{\frac{n-1}{2}} \not\equiv \left(\frac{a}{n}\right) \pmod{n} $$ then $n$ is strictly composite. $a$ is called an Euler witness. If the congruence holds, $n$ is a probable prime. It can be shown that if $n$ is composite, at least half of the elements in $\mathbb{Z}_n^*$ are Euler witnesses. Thus, $k$ iterations reduce the error probability to $2^{-k}$.
