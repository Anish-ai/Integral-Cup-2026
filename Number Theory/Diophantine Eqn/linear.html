1. Algebraic Foundations and Bézout's Identity

We commence our study of Diophantine equations with the linear case in two variables. The objective is to analyze the set of all integer pairs $(x, y) \in \mathbb{Z}^2$ that satisfy the equation $ax + by = c$, where $a, b, c \in \mathbb{Z}$ are given constants.

Theorem 1.1 (Solvability and Parameterization)

The linear Diophantine equation $ax + by = c$ has a solution in integers $x, y$ if and only if $d \mid c$, where $d = \gcd(a, b)$. If $(x_0, y_0)$ is any particular solution, then the entire infinite family of solutions is parameterized by:

$$x = x_0 + k\frac{b}{d}, \quad y = y_0 - k\frac{a}{d} \quad \text{for } k \in \mathbb{Z}$$

Let $S = \{ ax + by \mid x, y \in \mathbb{Z} \}$. By the Euclidean algorithm, the smallest positive integer in $S$ is precisely $d = \gcd(a, b)$, and thus $S$ is exactly the set of multiples of $d$. Therefore, $c \in S \iff d \mid c$.

Assume $d \mid c$ and let $(x_0, y_0)$ be a solution, so $ax_0 + by_0 = c$. Let $(x, y)$ be any other solution. Subtracting the two yields:

$$a(x - x_0) + b(y - y_0) = 0 \implies \frac{a}{d}(x - x_0) = -\frac{b}{d}(y - y_0)$$

Since $\gcd\left(\frac{a}{d}, \frac{b}{d}\right) = 1$, Euclid's Lemma dictates that $\frac{b}{d}$ must divide $(x - x_0)$. Hence, there exists an integer $k$ such that $x - x_0 = k\frac{b}{d}$. Substituting this back into the relation yields $y - y_0 = -k\frac{a}{d}$. It is trivial to verify that any choice of $k \in \mathbb{Z}$ produces a valid solution.

$\blacksquare$

1.1 Algorithmic Realization (Extended Euclidean Algorithm)

To compute the particular solution $(x_0, y_0)$, we employ the Extended Euclidean Algorithm (EEA) to find $x', y'$ such that $ax' + by' = d$, and then scale the result by $\frac{c}{d}$. Below is the aggressively optimized, theoretically minimal C++ implementation tailored for low-latency computational mathematics.

long long f(long long a,long long b,long long&x,long long&y){
if(!b){x=1;y=0;return a;}
long long u,v,d=f(b,a%b,u,v);
x=v;y=u-v*(a/b);
return d;
}
bool s(long long a,long long b,long long c,long long&x,long long&y){
long long d=f(abs(a),abs(b),x,y);
if(c%d)return 0;
x*=c/d;y*=c/d;
if(a<0)x=-x;if(b<0)y=-y;
return 1;
}

2. Advanced Addition: The Frobenius Coin Problem

A natural topological restriction on our solution space $\mathbb{Z}^2$ is to strictly demand $x, y \in \mathbb{Z}_{\ge 0}$. When $a, b > 0$ and $\gcd(a,b) = 1$, the question of which non-negative integers $c$ can be expressed as $ax + by$ is famously known as the Frobenius Coin Problem, or the Chicken McNugget Theorem.

Theorem 1.2 (Sylvester's Theorem, 1884)

Let $a, b \in \mathbb{Z}_{>0}$ such that $\gcd(a, b) = 1$.

  1. The largest integer $c$ that cannot be expressed in the form $ax + by$ for non-negative integers $x, y$ is $g(a,b) = ab - a - b$.
  2. The total number of non-expressible non-negative integers is exactly $\frac{(a-1)(b-1)}{2}$.

We prove (1). Consider the equation $ax + by = ab - a - b$. Assume for the sake of contradiction that a solution exists with $x \ge 0, y \ge 0$.

Taking the equation modulo $a$, we observe:

$$by \equiv -b \pmod a \implies b(y+1) \equiv 0 \pmod a$$

Since $\gcd(a,b) = 1$, we must have $y+1 \equiv 0 \pmod a$. Because $y \ge 0$, the minimal possible value for $y$ is $a - 1$.

By symmetry, analyzing modulo $b$ necessitates $x \ge b - 1$.

If we substitute these minimal bounds back into the original expression, we obtain:

$$ax + by \ge a(b-1) + b(a-1) = 2ab - a - b > ab - a - b$$

This strict inequality contradicts our initial assumption. Hence, $ab - a - b$ cannot be represented.

To show that every integer $N > ab - a - b$ is representable, observe that the integers $0, b, 2b, \dots, (a-1)b$ form a complete residue system modulo $a$. For any $N$, there exists some unique $0 \le y \le a-1$ such that $by \equiv N \pmod a$. Thus, $N - by = ax$ for some integer $x$. We must show $x \ge 0$.

$$ax = N - by > (ab - a - b) - b(a-1) = -a \implies ax \ge 0 \implies x \ge 0$$

Thus, $(x, y)$ provides a valid non-negative parameterization for any $N > ab - a - b$.

$\blacksquare$
