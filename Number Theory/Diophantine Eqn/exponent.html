1. Algebraic Factorization in the Exponent

When variables appear as exponents, bounding techniques shift from polynomial inequalities to growth rate analyses and parity-based factorizations. Consider the canonical introductory equation $n \cdot 2^n + 1 = y^2$ for $n, y \in \mathbb{N}$. We rearrange this via a difference of squares:

$$n \cdot 2^n = y^2 - 1 = (y - 1)(y + 1)$$

Since the difference between $(y+1)$ and $(y-1)$ is exactly $2$, they share the exact same parity. If they were odd, their product would be odd, contradicting the factor of $2^n$ (for $n \ge 1$). Thus, both are even. More rigorously, $\gcd(y-1, y+1) = \gcd(y-1, 2) = 2$. This implies that one factor contains almost the entire power of $2$, specifically $2^{n-1}$, while the other contains merely a single $2$. This structural bottleneck forces extreme constraints on $n$, reducing the search space to a tiny, finite set of test cases.

2. The Lifting The Exponent (LTE) Lemma

For competitive programmers and algebraic number theorists, the LTE Lemma is an indispensable engine for annihilating exponential Diophantine equations modulo prime powers. We first define the $p$-adic valuation $v_p(x)$, which denotes the exponent of the highest power of a prime $p$ that divides $x$.

Lemma 6.1 (LTE for Odd Primes)

Let $p$ be an odd prime. Let $x$ and $y$ be integers such that $p \nmid x$ and $p \nmid y$. If $p \mid (x - y)$, then for any positive integer $n$:

$$v_p(x^n - y^n) = v_p(x - y) + v_p(n)$$

We first prove the base case where $n = p$. Let $v_p(x - y) = k \ge 1$. We can write $x = y + c p^k$ where $p \nmid c$. Expand $x^p$ using the Binomial Theorem:

$$x^p - y^p = (y + cp^k)^p - y^p = \binom{p}{1} y^{p-1} cp^k + \binom{p}{2} y^{p-2} (cp^k)^2 + \dots + (cp^k)^p$$

The first term is $p \cdot y^{p-1} cp^k = c y^{p-1} p^{k+1}$. Since $p \nmid c$ and $p \nmid y$, the $p$-adic valuation of this first term is exactly $k + 1$.

We analyze the subsequent terms. The $j$-th term is $\binom{p}{j} y^{p-j} (cp^k)^j$. Its valuation is at least $1 + jk$. Because $p \ge 3$ and $j \ge 2$, we have $1 + jk > k + 1$. Since the valuation of a sum is strictly equal to the minimum of the valuations of its terms (if a unique minimum exists), we conclude:

$$v_p(x^p - y^p) = k + 1 = v_p(x - y) + 1$$

We extend this to any $n$ via induction. Write $n = p^\alpha \cdot m$ where $p \nmid m$. By standard algebraic factorization, $x^m - y^m = (x-y)(x^{m-1} + \dots + y^{m-1})$. Because $x \equiv y \pmod p$, the second factor is congruent to $m y^{m-1} \pmod p$. Since $p \nmid m$ and $p \nmid y$, $v_p(x^m - y^m) = v_p(x-y)$. Applying the base case $\alpha$ times sequentially strictly adds $\alpha = v_p(n)$ to the valuation, completing the proof.

$\blacksquare$
Lemma 6.2 (LTE for p = 2)

The prime $p=2$ is anomalous. Let $x$ and $y$ be odd integers.

  • If $n$ is even and $2 \mid (x - y)$, then: $v_2(x^n - y^n) = v_2(x - y) + v_2(x + y) + v_2(n) - 1$
  • If $n$ is odd and $4 \mid (x - y)$ (or if $4 \mid (x+y)$ with $+$ signs), the formula holds identically to the odd prime case.

2.1 Algorithmic Realization ($p$-adic Valuation)

To implement LTE-based pruning in computational searches, a blazing-fast valuation function is required.

long long v(long long p,long long n){
long long c=0;
while(n&&n%p==0){c++;n/=p;}
return c;
}

3. Zsigmondy's Theorem

When analyzing exponential Diophantine sequences like $a^n - b^n$, a recurring obstacle is determining when the sequence generates genuinely "new" prime factors. Zsigmondy's theorem resolves this with overwhelming topological certainty.

Theorem 6.3 (Zsigmondy, 1892)

Let $a > b > 0$ be coprime integers. Then for any integer $n \ge 1$, there exists at least one prime $p$ (called a primitive prime divisor) such that $p \mid (a^n - b^n)$ but $p \nmid (a^k - b^k)$ for all positive integers $k < n$, with exactly two rigid exceptions:

  1. $n = 6, a = 2, b = 1$. The sequence term is $2^6 - 1^6 = 63$. The prime factors are $3$ (which divides $2^2-1=3$) and $7$ (which divides $2^3-1=7$).
  2. $n = 2$, and $a + b$ is a power of 2.

In competitive programming, Zsigmondy's theorem guarantees that sequences like $2^n - 1$ inherently spawn new primes, allowing algorithms to safely assume the existence of a primitive root for virtually all exponents without explicit factorization checks.

4. Catalan's Conjecture (Mihăilescu's Theorem)

For over a century, a seemingly elementary question concerning perfect powers adjacent to each other remained unsolved: Are 8 ($2^3$) and 9 ($3^2$) the only consecutive non-trivial powers in existence?

Theorem 6.4 (Mihăilescu, 2002)

The exponential Diophantine equation $x^a - y^b = 1$ has exactly one solution for strictly positive integers $x, y, a, b > 1$, which is $3^2 - 2^3 = 1$.

The proof constructed by Preda Mihăilescu bypasses standard modular bounding and descends into the theory of cyclotomic fields. By analyzing the annihilators of ideal class groups modeled as Galois modules (specifically utilizing Stickelberger's theorem and Thaine's theorem), Mihăilescu proved that any solution aside from the canonical $3^2 - 2^3$ forces an algebraic contradiction regarding the size of the class group relative to the exponent bounds.
