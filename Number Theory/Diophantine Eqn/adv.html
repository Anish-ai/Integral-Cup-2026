Module VII: The Local-Global Principle

Algorithmic Number Theory & Abstract Algebra • Advanced Undergraduate Sequence

1. Topological Completions of the Rationals

To analyze Diophantine equations from an advanced algebraic geometry perspective, we shift our focus from $\mathbb{Z}$ to the field of rational numbers $\mathbb{Q}$. Ostrowski's Theorem asserts that every non-trivial absolute value on $\mathbb{Q}$ is equivalent to either the standard Euclidean absolute value (yielding the real numbers $\mathbb{R}$) or a $p$-adic absolute value for some prime $p$ (yielding the field of $p$-adic numbers $\mathbb{Q}_p$).

These completions ($\mathbb{R}$ and $\mathbb{Q}_p$) are termed "local fields". The Local-Global Principle postulates that for certain classes of equations, finding a "global" solution over $\mathbb{Q}$ is strictly equivalent to finding "local" solutions over all completions of $\mathbb{Q}$.

2. The Hasse-Minkowski Theorem

While the Local-Global principle fails for general higher-degree polynomials (e.g., Selmer's famous counterexample $3x^3 + 4y^3 + 5z^3 = 0$), it holds perfectly and rigorously for quadratic forms.

Theorem 7.1 (Hasse-Minkowski)

Let $Q(x_1, \dots, x_n)$ be a homogeneous quadratic form with coefficients in $\mathbb{Q}$. Then the equation $Q(x_1, \dots, x_n) = 0$ has a non-trivial solution in $\mathbb{Q}$ if and only if it has a non-trivial solution in $\mathbb{R}$ and in $\mathbb{Q}_p$ for every prime $p$.

The necessity ($\implies$) is trivial: since $\mathbb{Q} \subset \mathbb{R}$ and $\mathbb{Q} \subset \mathbb{Q}_p$, a global rational solution identically maps to a local solution in every completion. The profound power of the theorem lies in sufficiency ($\impliedby$): if no local obstructions exist, a global rational solution is mathematically guaranteed to exist.

3. Legendre's Ternary Quadratic Form

The most canonically accessible instantiation of the Hasse-Minkowski theorem resolves the solvability of the diagonal ternary quadratic form $ax^2 + by^2 + cz^2 = 0$.

Theorem 7.2 (Legendre's Theorem)

Let $a, b, c$ be non-zero, square-free integers that are pairwise relatively prime. The equation $ax^2 + by^2 + cz^2 = 0$ has a non-trivial solution in integers $(x,y,z)$ if and only if:

  1. $a, b, c$ do not all have the same sign (Solvability over $\mathbb{R}$).
  2. $-ab$ is a quadratic residue modulo $c$. (Solvability over $\mathbb{Q}_c$)
  3. $-bc$ is a quadratic residue modulo $a$. (Solvability over $\mathbb{Q}_a$)
  4. $-ca$ is a quadratic residue modulo $b$. (Solvability over $\mathbb{Q}_b$)

We prove the necessity of condition (3) via modular reduction, which models the $p$-adic obstruction. Assume a non-trivial integer solution $(x,y,z)$ exists. We can divide out common factors to assume $\gcd(x,y,z) = 1$.

Consider the equation modulo $a$. Since $a$ divides $ax^2$, we have:

$$by^2 + cz^2 \equiv 0 \pmod a$$

Assume for contradiction that $\gcd(y, a) > 1$. Let $p$ be a prime dividing both $y$ and $a$. Then $p \mid by^2$. Since $p \mid a$, the original equation $ax^2 + by^2 + cz^2 = 0$ implies $p \mid cz^2$. Since $a, b, c$ are pairwise coprime, $p \nmid c$. Thus $p \mid z^2 \implies p \mid z$. But if $p \mid a$, $p \mid y$, and $p \mid z$, we substitute back into $ax^2 = -by^2 - cz^2$. Because $a$ is square-free, its $p$-adic valuation is 1. The right side has $p$-adic valuation $\ge 2$. This strictly forces $p \mid x$, contradicting $\gcd(x,y,z)=1$. Therefore, $\gcd(y,a) = 1$.

Since $y$ is coprime to $a$, its modular inverse $y^{-1}$ exists modulo $a$. We multiply the congruence by $b (y^{-1})^2$:

$$b^2 + bc (z y^{-1})^2 \equiv 0 \pmod a \implies (z y^{-1} b)^2 \equiv -b^2 c \equiv -bc \cdot b \pmod a$$

Multiplying by $c$ (which is valid since $\gcd(c,a)=1$), we isolate the residue:

$$-bc \equiv (z y^{-1} c)^2 \pmod a$$

This explicitly demonstrates that $-bc$ is a quadratic residue modulo $a$. By pure symmetry, the same holds for the other cyclic permutations. The sufficiency proof constructs the solution via Minkowski's Geometry of Numbers or by gluing local $p$-adic solutions via Hensel's Lemma, ensuring the local conditions lift to a global intersection.

$\blacksquare$

3.1 Algorithmic Realization (Legendre's Conditions)

To computationally determine if a ternary quadratic form is solvable, we evaluate the Legendre symbols. The following C++ implementation rapidly computes the solvability for pairwise coprime, square-free integers using Euler's criterion for the Jacobi/Legendre symbol.

long long r=1;b=(b%m+m)%m;
while(e){if(e&1)r=r*b%m;b=b*b%m;e>>=1;}
return r;
}

if(p_==1)return 1;
long long r=p(n,(p_-1)/2,p_);
return r<=1;
}
bool s(long long a,long long b,long long c){
if((a>0&&b>0&&c>0)||(a<0&&b<0&&c<0))return 0;
return q(-a*b,abs(c))&&q(-b*c,abs(a))&&q(-c*a,abs(b));
