Module 1: Divisibility and Primes

Part C: Greatest Common Divisor

1. Bézout's Identity

The greatest common divisor of two integers is not merely a number that divides both; it is fundamentally the smallest positive linear combination of the two integers. This algebraic property is formalized in Bézout's Identity.

Theorem 1.1 (Bézout's Identity)

For any non-zero integers $a$ and $b$, let $d = \gcd(a, b)$. Then there exist integers $x$ and $y$ such that:

$$ax + by = d$$

Furthermore, $d$ is the smallest positive integer that can be expressed in this form.

Proof

Consider the set $S$ of all strictly positive linear combinations of $a$ and $b$:

$$S = \{ au + bv > 0 \mid u, v \in \mathbb{Z} \}$$

$S$ is non-empty because taking $u = a$ and $v = b$ yields $a^2 + b^2 > 0$ (since $a, b \neq 0$). By the Well-Ordering Principle, $S$ must contain a smallest element, which we will call $d$. Thus, $d = ax + by$ for some specific $x, y \in \mathbb{Z}$.

We must show that $d = \gcd(a, b)$. First, we show $d \mid a$. By the Division Algorithm, there exist integers $q, r$ such that $a = dq + r$, with $0 \le r < d$. Substituting our expression for $d$ gives:

$$r = a - dq = a - (ax + by)q = a(1 - xq) + b(-yq)$$

If $r > 0$, then $r$ is a strictly positive linear combination of $a$ and $b$, meaning $r \in S$. However, $r < d$, which contradicts the assumption that $d$ is the smallest element in $S$. Thus, $r = 0$, implying $d \mid a$. A symmetric argument shows $d \mid b$. Therefore, $d$ is a common divisor.

Next, let $c$ be any arbitrary common divisor of $a$ and $b$. Then $c \mid a$ and $c \mid b$, so $c$ must divide any linear combination of $a$ and $b$. Specifically, $c \mid (ax + by)$, meaning $c \mid d$. Since $d > 0$, it follows that $c \le d$. Thus, $d = \gcd(a, b)$. $\blacksquare$

2. The Extended Euclidean Algorithm

Bézout's Identity guarantees the existence of coefficients $x$ and $y$, but the Extended Euclidean Algorithm provides a constructive method to find them in $O(\log(\min(a,b)))$ time. This is accomplished by algebraically unrolling the sequence of remainders from the standard Euclidean Algorithm.

Consider the recursive step of the Euclidean Algorithm: $\gcd(a, b) = \gcd(b, a \bmod b)$. Assume the recursive call successfully finds coefficients $u, v$ for the smaller problem:

$$b \cdot u + (a \bmod b) \cdot v = \gcd(a, b)$$

We know that $a \bmod b = a - b \lfloor \frac{a}{b} \rfloor$. Substituting this into the equation yields:

$$b \cdot u + \left(a - b \lfloor \frac{a}{b} \rfloor\right) \cdot v = \gcd(a, b)$$ $$a \cdot v + b \left(u - v \lfloor \frac{a}{b} \rfloor\right) = \gcd(a, b)$$

Comparing this to our target form $ax + by = \gcd(a, b)$, we can extract the coefficient transitions directly:

$$x = v, \quad y = u - v \lfloor \frac{a}{b} \rfloor$$

The corresponding C++ implementation:

long long f(long long a, long long b, long long &x, long long &y) {
    if (!b) {
        x = 1;
        y = 0;
        return a;
    }
    long long u, v;
    long long d = f(b, a % b, u, v);
    x = v;
    y = u - v * (a / b);
    return d;
}

3. Properties of Relatively Prime Integers

Definition 3.1 (Coprimality)

Two integers $a$ and $b$ are said to be relatively prime (or coprime) if their greatest common divisor is $1$, i.e., $\gcd(a, b) = 1$.

From Bézout's Identity, an immediate and powerful equivalence follows:

Theorem 3.2

Integers $a$ and $b$ are relatively prime if and only if there exist integers $x, y$ such that $ax + by = 1$.

This algebraic formulation effortlessly yields classical divisibility theorems, notably the extension of Euclid's Lemma.

Theorem 3.3 (General Euclid's Lemma)

If $a \mid bc$ and $\gcd(a, b) = 1$, then $a \mid c$.

Proof

Since $\gcd(a, b) = 1$, by Bézout's Identity there exist integers $x, y$ such that $ax + by = 1$. Multiply the entire equation by $c$:

$$acx + bcy = c$$

Notice that $a$ obviously divides $acx$. Furthermore, since we are given that $a \mid bc$, it must be true that $a \mid (bc)y$. Because $a$ divides both terms on the left side of the equation, it must divide their sum. Thus, $a \mid (acx + bcy) \implies a \mid c$. $\blacksquare$

4. Advanced Addition: The GCD of Fibonacci Numbers

The sequence of Fibonacci numbers $F_n$ interacts beautifully with divisibility properties, yielding a striking closure theorem: the greatest common divisor of two Fibonacci numbers is itself a Fibonacci number.

Theorem 4.1 (Fibonacci GCD Theorem)

For any positive integers $m$ and $n$, $\gcd(F_m, F_n) = F_{\gcd(m, n)}$.

Proof

We first establish a critical identity regarding the addition of indices. By induction on $n$, one can prove:

$$F_{m+n} = F_{m+1}F_n + F_m F_{n-1}$$

Next, we establish that consecutive Fibonacci numbers are coprime, i.e., $\gcd(F_k, F_{k-1}) = 1$. This is trivially true via the Euclidean algorithm, since $F_k = 1 \cdot F_{k-1} + F_{k-2}$, implying $\gcd(F_k, F_{k-1}) = \gcd(F_{k-1}, F_{k-2})$. Descending to the base cases yields $\gcd(1, 1) = 1$.

Assume without loss of generality that $m \ge n$. Let us rewrite $m$ as $m = n + (m-n)$. Applying the addition identity yields:

$$F_m = F_{(m-n)+n} = F_{m-n+1}F_n + F_{m-n}F_{n-1}$$

We want to find $\gcd(F_m, F_n)$. By substituting the expression for $F_m$, we get:

$$\gcd(F_m, F_n) = \gcd(F_{m-n+1}F_n + F_{m-n}F_{n-1}, F_n)$$

Since any multiple of $F_n$ vanishes modulo $F_n$, the term $F_{m-n+1}F_n$ can be eliminated, yielding:

$$\gcd(F_m, F_n) = \gcd(F_{m-n}F_{n-1}, F_n)$$

We know that consecutive Fibonacci numbers are coprime: $\gcd(F_{n-1}, F_n) = 1$. Therefore, multiplying by $F_{n-1}$ does not introduce any new common divisors with $F_n$. By Theorem 3.3, we can drop $F_{n-1}$ entirely:

$$\gcd(F_m, F_n) = \gcd(F_{m-n}, F_n)$$

Notice that this relation mirrors the recursive subtraction step of the Euclidean algorithm applied to the indices $m$ and $n$. Just as $\gcd(m, n) = \gcd(m-n, n)$ repeatedly descends until reaching $\gcd(d, 0) = d$ (where $d = \gcd(m,n)$), the expression $\gcd(F_m, F_n)$ maps perfectly to the same subtraction sequence on the indices.

The recursion will trace the Euclidean algorithm on $m, n$ until it hits $\gcd(F_{\gcd(m,n)}, F_0)$. Since $F_0 = 0$, we have:

$$\gcd(F_m, F_n) = F_{\gcd(m,n)}$$

The theorem is thus proven. $\blacksquare$
